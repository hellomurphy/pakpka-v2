# 📱 Tenant Portal (The Spoke)

**A frictionless, mobile-first experience for tenants integrated via LINE LIFF.**

## 🛠 Technical Highlights

- **Platform:** Integrated with **LINE Official Account (OA)** using **LIFF SDK**.
- **Authentication:** Passwordless onboarding via **LINE OAuth2** and phone number identity matching.
- **UX/UI:** Optimized for mobile-first interactions, focusing on bill payment and digital receipt management.
- **Real-time Sync:** Lightweight client-side application that communicates with the Admin Hub for instant data updates.

## 🚀 Development

```bash
# Run standalone tenant portal
pnpm --filter client dev
```
