# 🌐 Marketing & Landing Page

**High-performance, SEO-optimized entry point for the PMS platform.**

## 🛠 Technical Highlights

- **Edge Delivery:** Deployed on **Cloudflare Workers** for zero-latency global delivery and sub-second Time to First Byte (TTFB).
- **SEO Optimized:** Utilizes Nuxt 4's native SSR capabilities for maximum search engine visibility.
- **Conversion Focused:** Minimalist design with integrated lead capture and clear pricing transparency.

## 🚀 Development

```bash
# Run standalone landing page
pnpm --filter landing dev
```
